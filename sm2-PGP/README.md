### SM2—PGP   

### 算法思路：   
    加密阶段：首先要随机选取一个密钥K，用于对称密码的加密。利用SM2的公钥加密K，发送给接收者，然后用密钥K加密明文。     
    
    解密阶段：接收者利用私钥解密出密钥K，从而解密出明文。
