
## SM3 birthday attack   

### 基本思想：                        
       首先我们要实现sm3加密算法，这一部分我们参考相应的国密加密标准即可。生日攻击的目的是找到两个明文m,m'使得hash(m)=hash(m')      
       我们知道sm3输出的杂凑值是64字节，即256bit.根据生日攻击理论。为了满足碰撞要求，至少需要1.17*2^128次。显然搜索的过程是非常     
       缓慢的。因此笔者这里寻找的是两个hash前8bit相同的碰撞。
